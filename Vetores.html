Vetores

Vetor é um - subtipo de dado - (tipo: object) que comporta vários valoresm acessíveis por meio de sua posição dentro do conjunto.


1 - Vetores em JS não têm tamanho predefinido.
Como declarar um vetor vazio em JS

let vet1 = Array(); // Forma antiga
let vet2 = []; // Forma nova

Como delcarar vetor com conteúdo:

let vet3 = Array(44, 'Banana', true, undefined); //Antiga
let vet4 = [44, 'Banana', true, undefined]; //Nova

2 - Diferentes tipos de dados podem ser colocados dentro do mesmo vetor.

3 - TODOS os vetores têm suas posições iniciadas em 0 (zero).

vet3

  44 | 'Banana' | true | undefined

.. 0..| ........1......| ...2...|.......3....

alert(vet3[0]);

Adicionando um valor a um vetor.

push() => adiciona um valor ao final do vetor.

vet2.push('Oi'); // Coloca o valor 'Oi' na pos. 0
vet4.push('Oi'); // Coloca o valor 'Oi' na pos. 4

Descobrindo quantos elementos há em um vetor: length

vet1.length // 0
vet2.length // 1
vet3.length // 4
vet4. length // 5

FIFO First In, First Out Fila

FILO First in, Last Out Pilha

Percorrendo o vetor.

for (let x = 0; x < vet3.length; x++) {
     document.write(vet3[x]);
}

Retirando elementos do final do vetor: pop()

let val = vet3.pop();
